What is a RTTEX

An RTTEX file is a texture file saved in the Robinson Technologies Texture (RTTEX) format. It is used by Growtopia, a game that allows you to build worlds, compose music, choose an occupation, trade items, and earn achievements.


How to use

Step 1 : Prepare an rttex file.

Step 2 : Click the "Convert" button.

Step 3 : Select the file to be converted.

Step 4 : If an image appears, it means it has been converted.

Step 5 : If you want to save the image, click on the image.


Usage

Converting .rttex files to .png and .png to .rttex
